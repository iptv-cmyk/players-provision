1.1
---
Added support to configure accessibility settings.
Usage:
- python prov.py -a <IP_ADDRESS> to enable HomeButtonAccessibilityService on a specific IP.
- Automatic configuration of the accessibility service during the standard provisioning/installation flow.
- Added support to install/uninstall a specific box IP.
  Usage: python prov.py -i <IP_ADDRESS> or python prov.py -u <IP_ADDRESS>
