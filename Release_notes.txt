1.4
---
* Added support to verify Netflix APKs (both single package and split package directories prefixed with 'netflix_') using apksigner.
* Added support to deploy/install single or split Netflix packages across all subnet nodes or to a specific target IP via `--install-netflix`.

1.3
---
* Added scan mode (-s, --scan) to check available boxes on the network and show their device names/models as well as their Netflix installation status and version.

1.2
---
* Verbose support (-v)
* new launcher version (10.3).

1.1
---
Added support to configure accessibility settings.
Usage:
- python prov.py -a <IP_ADDRESS> to enable HomeButtonAccessibilityService on a specific IP.
- Automatic configuration of the accessibility service during the standard provisioning/installation flow.
- Added support to install/uninstall a specific box IP.
  Usage: python prov.py -i <IP_ADDRESS> or python prov.py -u <IP_ADDRESS>
